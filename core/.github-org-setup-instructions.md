# GitHub Organization Profile Setup Instructions

## How to Add This Profile to Your GitHub Organization

### Step 1: Create the `.github` Repository

1. Go to your GitHub organization: `https://github.com/fabrknt`
2. Click **"New repository"**
3. Name it exactly: `.github` (with the dot)
4. Make it **Public**
5. Add a description: "GitHub organization profile"
6. Click **"Create repository"**

### Step 2: Create the Profile Directory Structure

In the `.github` repository, create this structure:

```
.github/
└── profile/
    └── README.md
```

### Step 3: Add the Profile Content

1. Create a folder named `profile` in the `.github` repository
2. Inside the `profile` folder, create a file named `README.md`
3. Copy the contents from `.github-org-profile.md` into `profile/README.md`
4. Commit and push

### Step 4: Verify

1. Go to your organization page: `https://github.com/fabrknt`
2. The profile README should now display at the top of the page

---

## Alternative: Quick Setup via GitHub Web Interface

1. Go to `https://github.com/fabrknt`
2. Click **"New repository"**
3. Repository name: `.github`
4. Check **"Add a README file"**
5. Click **"Create repository"**
6. In the `.github` repo, click **"Add file"** → **"Create new file"**
7. Name the file: `profile/README.md` (GitHub will auto-create the `profile` folder)
8. Paste the contents from `.github-org-profile.md`
9. Commit the file

---

## Alternative: Quick Setup via Command Line

```bash
# Clone the .github repository (or create it first on GitHub)
git clone git@github.com:fabrknt/.github.git
cd .github

# Create profile directory
mkdir -p profile

# Copy the profile content
cp /path/to/.github-org-profile.md profile/README.md

# Commit and push
git add profile/README.md
git commit -m "Add organization profile README"
git push origin main
```

---

## Tips

- The repository **must** be named exactly `.github` (with the dot)
- The file **must** be at `profile/README.md`
- The repository **must** be public for the profile to display
- Changes appear immediately after pushing
